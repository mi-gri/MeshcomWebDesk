# Release Notes – v1.12.2

## What is new in v1.12.2

---

## 🔍 QRZ First Name Always Visible

The first name from QRZ lookup is now displayed **directly next to the callsign** in the chat and monitor – no hover required.

---

## 🤖 Bot: `--ping` and `--version` Work When Bot Is Disabled

The `--ping` and `--version` commands are now accepted **even when the bot is turned off**. Useful for quick reachability checks without enabling the full bot.

---

## 🔐 Console Command Helper: Password Fields Masked

The `--passwd` and `--setpwd` fields are now rendered as **password inputs** (`type=password`) in the Console Command Helper. Password values no longer appear in plain text in command badges or toast notifications.

---

## 🔋 Console Command Helper: Battery Voltage & Capacity

- Parsing of **voltage (V) and percentage (%)** from node status output
- Visual **progress bar** for battery charge level
- StatusCommand improvements: optimistic update on change, global parse index to prevent value reversion

---

## 📅 Telemetry Send Times Improved

- Limit increased to **24 entries**
- **`*`** wildcard to match all hours
- Validation hints for duplicate and invalid entries

---

## 🔧 Bug Fixes

- **Bot / auto-reply: minimum 500 ms delay** – prevents node UDP crash caused by replies sent too quickly after receiving a packet
- **Send button disabled at packet limit** – the send button is locked when the message text exceeds the firmware JSON packet size limit
- **Emoji-heavy messages: firmware packet size** – multi-byte emojis (4 bytes UTF-8) are now correctly counted when calculating packet size
- **Multi-node: new nodes register without app restart** – newly configured nodes become active immediately without restarting the service
- **Device / listen port fields are read-only** – port fields in the Connection and Additional Nodes settings are now read-only with an explanatory hint
- **Enter key sends message** – line breaks now require Shift+Enter; plain Enter sends
- **AI search all-contacts mode: 4× message limit** – older search results are no longer cut off when searching across all direct QSOs
- **VersionCommand: removed LI prefix** – formatting artefact stripped from the version string output
- **Mobile: licence badge and map search field** – improved layout on small screens (≤ 600 px); PayPal donation hint hidden on iPhone
