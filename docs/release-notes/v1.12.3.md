# Release Notes – v1.12.3

## What is new in v1.12.3

---

## 🔧 Bug Fixes & Improvements

---

## 🔍 AI Search: Smarter Token Management

- Messages are trimmed by estimated token count to stay within model limits and avoid rate-limit errors.
- A warning is shown when older messages were excluded due to the token budget, with a **"Search in previous period"** button to continue the search further back in time.
- Bot commands and JSON system messages are now excluded from the AI search and summary context to reduce noise.
- The `Max. messages` setting now applies to QSO summaries only; AI search uses the selected time period as its primary constraint.
- Date inputs replaced with a reusable **DatePicker** component.

---

## 🖥️ Console: MH List & Help Popup

The Console Command Helper now shows a **Most-Heard (MH) list** popup and a **Help** popup directly within the console view.

---

## 🔌 NET Console

The console page now uses **NET Console** (plain TCP) instead of TLS. The TLS option has been removed from settings.

## 🐛 Bug Fixes

- **Echo-timeout warning** – cleared correctly when a relay echo arrives from a sibling node.
- **Telnet line buffer wrap** – console lines are now processed correctly when the internal line buffer wraps around.
- **Linux / Docker: permission error on startup** – a clear error message is now shown when the data or log directory cannot be created due to missing write permissions, instead of a silent crash.
- **Docker: missing writable directories** – `/app/data` and `/app/keys` are now pre-created in the image so the app starts cleanly when running as a non-root user.
- **WeatherApi: TelemetryFilePath write error** – an `UnauthorizedAccessException` when writing the telemetry JSON file (e.g. path pointing to `/app/data`) is now caught and logged; the app continues running and shows the error in the status view.
- **Bot: empty reply for `{telemetry}`** – when `{telemetry}` cannot be resolved (file missing or path not configured), the bot now replies `(keine Daten)` instead of sending nothing.
